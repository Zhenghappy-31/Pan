MySQL Project: Data Cleaning & Exploratory Data Analysis

  • Project Page

Project Description:

In this project, I focused on cleaning a dataset related to company layoffs and performed exploratory data analysis (EDA) using MySQL. The dataset included multiple columns such as company, location, industry, total laid off, percentage laid off, date, and other attributes. Hereâ€™s a breakdown of the steps I followed:

1. Data Cleaning:

  • Duplicate Data: I used the ROW_NUMBER() function to identify and remove duplicate entries based on multiple columns, such as company name, location, and industry. After identifying the duplicates, I created a clean staging table where only unique records were retained.
  • Standardizing Data: I standardized columns by trimming unnecessary spaces from company names and cleaning up industry labels. For example, I standardized all variations of "Crypto" into one label. Similarly, I cleaned the country field by removing extra characters like periods and formatting the date column into a proper DATE type.
  • Fixing Null and Blank Values: To handle missing or blank values in certain fields like the industry column, I cross-referenced data from the same company and location to fill in the missing fields wherever possible.
  • Removing Unreliable Data: I deleted rows where crucial data like total laid off or percentage laid off was missing or unreliable, ensuring only relevant records were retained for analysis.

    • 2. Exploratory Data Analysis (EDA)

  • Date Range: I determined the date range of the layoffs data by finding the minimum and maximum dates from the cleaned table.
  • Most Impacted Location: I calculated the total number of layoffs for each location and identified the locations with the highest total layoffs.
  • Average Layoff Percentage by Industry: I calculated the average percentage of layoffs and the average total laid off for each industry. Additionally, I computed the total number of employees per industry based on these averages.
  • Rolling Monthly Layoff Total: I calculated the rolling total number of layoffs for each month using window functions to observe trends over time.
  • Top 5 Companies with Largest Layoffs Each Year: I ranked companies by their total layoffs for each year and identified the top 5 companies with the largest layoffs annually.

    • This project improved my skills in SQL query optimization, data cleaning, and using advanced SQL functions like window functions, CTE (Common Table Expressions), and aggregate functions to handle complex datasets. It also helped me draw valuable insights through exploratory data analysis, which could be beneficial for decision-making.
