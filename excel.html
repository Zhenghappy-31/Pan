Excel Project: Bike Sales Data Cleaning, Pivot Table Analysis, and Dashboard Creation

  • Project Page

Project Description:

In this project, I worked with a dataset of bike sales, where I performed data cleaning, created a pivot table for analysis, and developed a dashboard to visualize key metrics.

1. Data Cleaning:

  • Removing Duplicates: Ensured data accuracy by eliminating duplicate entries.
  • Handling Missing Values: Imputed missing values or excluded them based on their impact on analysis.
  • Standardizing Data: Corrected inconsistencies in data formats and values for consistent analysis.

    • 2. Pivot Table Creation:

  • Summarized data: Created pivot tables to aggregate and analyze bike sales data based on various dimensions (marital status, region, education, income, age bracket, and commute distance).
  • Provided insights: The pivot tables revealed the distribution of bike sales across different demographics and factors, identifying trends and patterns.

    • 3. Dashboard Development:

  • Visualized data: Created a dashboard to visually represent the findings from the pivot tables.
  • Included charts: Incorporated bar charts and line charts to illustrate the relationships between different variables and bike sales.
  • Provided interactivity: Implemented slicers to allow for dynamic filtering and analysis of the data based on different criteria.

    • 4. Findings and Insights:

  • Customer Segmentation: Identified key customer segments based on demographics and purchase behavior.
  • Marketing Opportunities: Determined that targeting customers with graduate degrees and higher incomes residing in urban areas with longer commute distances could be a promising marketing strategy.

    • Through this project, I strengthened my Excel skills in data cleaning, pivot table analysis, and dashboard creation, while also developing a better understanding of how to present data-driven insights to stakeholders in an intuitive format.
